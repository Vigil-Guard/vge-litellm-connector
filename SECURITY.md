# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** [contact@vigilguard.ai](mailto:contact@vigilguard.ai)

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgement:** within 48 hours
- **Initial assessment:** within 5 business days
- **Fix and disclosure:** coordinated with reporter

## Scope

This policy covers the `vge-litellm-connector` adapter service:

- Source code in this repository
- Docker image built from this repository
- Configuration files (docker-compose, litellm-config)

Out of scope:

- LiteLLM Proxy itself (report to [LiteLLM](https://github.com/BerriAI/litellm))
- Vigil Guard Enterprise API (report to contact@vigilguard.ai separately)
